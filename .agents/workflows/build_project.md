# Build Project Workflow

This workflow describes the steps required to install dependencies and build this Next.js project.

## 1. Install Dependencies
Ensure you have Node.js 20+ installed. Run the following command to install all NPM packages:

```bash
// turbo
npm install
```

## 2. Build Project
To compile and build the production-ready application, use:

```bash
// turbo
npm run build
```
